# Financial Data & Strategy Lab

This is my personal workspace for learning financial data analysis and trading.

I believe it's better to understand the data and calculations myself first, before depending on AI tools. So I split my work into two parts.

### 1. Main Focus: Manual Weekly Stock Tracker
**File**: weekly_stock_tracker_manual.xlsx  
**Status**: Active (I update it every week)

- I manually enter weekly stock data for big tech companies (Apple, Microsoft, Nvidia, Google, Meta).
- I calculate SMA_3, SMA_5, weekly returns, and trends myself.
- This helps me learn Excel deeply and understand market movements.

### 2. AI Project (Paused)
**File**: Stock_Data_LIVE.xlsx  
**Status**: Paused

I built this using AI to automatically pull stock data. It is kept here as proof that I know how to use AI, but right now I am focusing only on manual work to build strong basics.
